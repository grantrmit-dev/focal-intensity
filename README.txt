	  Program for the calculation of electric field in the focal region of a high
          numerical-aperture objective using the vectorial Debye integral 
	______________________________________________________________________________


** Contents

	1. Introduction
	2. Requirements
	3. How to use the programs
	

1. Introduction

	This software calculates the electric field near the focus of a lens of
	microscope objective according to the Chapter 2 of the book.


2. Requirements

	MATLAB 6.0 or newer.
	At least 256MB available RAM.

	
3. How to use the programs

	As a first step, you may set the system parameters in the "systempra.m" function.
	Switch to the 'Chapter2' folder (or make it part of the MATLAB path) and run the 
        "mainprogramme.m" function which will give the calculation results and plot it as
        the same style shown in the book figures.
